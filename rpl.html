Deskripsi Program

Program keahlian RPL membekali siswa dengan kompetensi dalam pengembangan perangkat lunak, termasuk pemrograman desktop, web, mobile, dan database. Siswa juga mempelajari metodologi pengembangan perangkat lunak, sistem operasi, dan pengujian perangkat lunak.

Kompetensi yang Diajarkan

  • Pemrograman Web (HTML, CSS, JavaScript, PHP, dll)
  • Pemrograman Aplikasi Mobile (Flutter, Kotlin, dll)
  • Pengembangan Database dan SQL
  • Pengujian dan Debugging Software
  • Manajemen Proyek Perangkat Lunak

Peluang Karier

Lulusan RPL dapat bekerja sebagai Web Developer, Mobile Developer, Software Tester, Database Administrator, atau melanjutkan studi ke jenjang yang lebih tinggi di bidang Teknologi Informasi.
